# Experiencia Interactiva de Cumple (Responsiva)

Estructura básica creada para una experiencia web responsiva con las siguientes pantallas:
- Pantalla de acceso (teclado numérico) — contraseña `21062000`
- Pastel animado con vela y música
- Carta animada con efecto de escritura
- Galería horizontal con fotos y un video final
- Despedida y botón `BYE` para volver al inicio

Cómo probar localmente:
1. Coloca los recursos listados en `assets/` (véase `assets/README.md`).
2. Abre `index.html` en un navegador (Chrome/Edge/Firefox). Para reproducir audio automáticamente puede ser necesario interactuar primero (click) por políticas del navegador.

Archivos clave:
- [index.html](index.html)
- [styles.css](styles.css)
- [script.js](script.js)

Si quieres que lo despliegue en un servidor ligero, puedes usar (Node) `npx serve` o `python -m http.server 8000`.
