Coloca los siguientes recursos en la carpeta `assets/` del proyecto para que la experiencia funcione plenamente:

- `background-music.mp3` — pista alegre y libre de derechos (ej: Pixabay Audio / Mixkit)
- `candle-sizzle.mp3` — sonido corto de vela/humo (Freesound / ZapSplat)
- `paper-open.mp3` — efecto de abrir sobre (Freesound)
- `confetti-sound.mp3` — (opcional) sonido de confeti
- `cake.gif` — GIF animado del pastel (se mostrará en la etapa del pastel)
- `overlay.png` — imagen que se colocará en la esquina inferior izquierda del GIF (opcional)
- `envelope.png` — imagen del sobre que se mostrará como modal al hacer click en LISTO
 - `letter-bg.jpg` — imagen de fondo para la página de la carta (letter.html)
 - `typing.mp3` — sonido corto de escritura para la animación de la carta
- `video-felicitation.mp4` — tu video de felicitación (mp4 preferible)
- `preview.jpg` — miniatura para el cuadro de video
- `img1.jpg`, `img2.jpg`, `img3.jpg` — imágenes para la galería (tamaños proporcionales, p. ej. 1280×720)

Notas:
- Usa archivos libres de derechos o que tengas permiso para usar. Las fuentes sugeridas: Freesound, Pixabay Audio, Mixkit, ZapSplat.
- Si no subes algún recurso, el script intentará continuar y mostrará advertencias en la consola.
